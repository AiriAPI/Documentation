# Generate Fact

Generate random anime Waifu from a huge database.


## Functions

| Function |   Type   | Description                                          | Required? |
| :------: | :------: | ---------------------------------------------------- | :-------: |
| getWaifu  | function | The search function to generate a random anime Waifu. |    Yes    |
