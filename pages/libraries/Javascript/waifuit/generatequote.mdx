# Generate Fact

Generate random anime Quote from a huge database.


## Functions

| Function |   Type   | Description                                          | Required? |
| :------: | :------: | ---------------------------------------------------- | :-------: |
| getQuote  | function | The search function to generate a random anime quote. |    Yes    |
